GDW GAME

Multilayer Instructions:
1. Connect all players through LAN via Ethernet Cable

2. Run Server.exe

3. Type in "ipconfig" into the command console

4. get the IPv4 Address of the Ethernet connection

5. Send that to each player who wants to join to input into the IP box in the game

6. The RTS Player must connect first

7. The FPS Players must connect after RTS Player has loaded in
